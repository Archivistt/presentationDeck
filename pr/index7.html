System Design & Architecture

MERN Stack Core

Utilizes React.js for the dynamic frontend, Node.js/Express.js for the robust API layer, and MongoDB for scalable real-time read/write data operations.

RFID Integration

Implements a "Tap-to-Verify" mechanism that instantly links each student's physical RFID card UID to their specific meal credit account.

Data Integrity

Maintains immutable, auditable transaction logs that record every single tap event, ensuring 100% accountability and accurate reporting.

Role-Based Access Control

Enforces strict security with distinct access levels and dashboards for PSAS administrators (monitoring) and canteen operators (service).

Data Flow Architecture

2
