Theoretical & Technical Framework

RFID Technology

Contactless Radio Frequency Identification enables fast, accurate, and touchless student verification at the point of service.

MERN Stack

Utilizes MongoDB, Express.js, React.js, and Node.js to provide a robust, scalable full-stack web development environment.

ISO/IEC 25010

International software quality model used to strictly evaluate characteristics such as performance efficiency, security, and usability.

Technology Acceptance Model (TAM)

A theoretical framework used to assess user acceptance by measuring Perceived Ease of Use and Perceived Utility.

Framework Integration

This study synergizes these pillars into a cohesive solution:

RFID handles hardware-level identity verification, the MERN Stack powers the web system, ISO/IEC 25010 sets the quality benchmarks, and TAM validates user adoption.
